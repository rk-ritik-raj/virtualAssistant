# AI Virtual Assistant 🤖

A full-stack AI-powered Virtual Assistant built using the MERN Stack that enables users to interact through both voice and text. The assistant uses Google Gemini AI to generate intelligent responses and supports voice conversations using the Web Speech API. Users can securely create accounts, customize their assistant, and upload personalized assistant images.

## 🚀 Features

* 🎤 Voice Input using Web Speech API
* 🔊 Voice Output (Text-to-Speech)
* 🧠 Smart AI Responses powered by Gemini AI
* 🔐 Secure Authentication using JWT & bcryptjs
* 👤 User Registration and Login System
* 🖼️ Upload Custom Assistant Images
* ✏️ Customize Assistant Name and Branding
* ☁️ Cloud Image Storage using Cloudinary
* 📱 Fully Responsive Design
* 🌐 Deployed and Accessible Online

## 📌 How It Works

1. User creates an account or logs in.
2. User can customize the assistant's name and image.
3. User interacts with the assistant using voice or text.
4. Speech input is converted to text using the Web Speech API.
5. The text is sent to Gemini AI for processing.
6. Gemini AI generates an intelligent response.
7. The response is displayed on the screen and spoken back to the user using text-to-speech.
